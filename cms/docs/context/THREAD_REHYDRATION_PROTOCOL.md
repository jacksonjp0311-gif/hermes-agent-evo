# Thread Rehydration Protocol

CMS-SA v0.4.0

A fresh AI-agent thread is not version-ready until it completes three scans:

## 1. Origin Theory Scan

Reads the founding theory and reflection surfaces.

Purpose: recover why the system exists.

Required targets:

- `docs/theory/`
- `docs/reflection/law_of_sufficient_form_v0_2b3.md`
- root README current public finding
- current non-claim locks
- current feedback / memory governance rules

## 2. Software Architecture Scan

Reads the architecture genesis layer and runtime contracts.

Purpose: recover how the theory became executable software.

Required targets:

- `docs/architecture/cms_sa_v0_1_software_architecture.tex`
- `docs/architecture/`
- `reports/architecture/`
- `src/cms/README.md`
- `AGENTS.md`
- `README_90_SECONDS.md`
- `docs/context/repository_context_index.json`
- `docs/context/rcc_nexus_index.json`
- `rcc/nexus/route_map.json`
- `rcc/nexus/task_routing_matrix.md`

## 3. Runtime State Scan

Reads the current live repo state.

Purpose: recover where the system is now.

Required targets:

- `README.md`
- `outputs/version_registry/cms_version_registry.json`
- `reports/public_sync/latest_public_sync_report.md`
- `reports/geometry/latest_reflective_git_geometry_validation.md`
- `reports/feedback/latest_feedback_lifecycle_validation.md`
- `reports/alignment/latest_multilevel_alignment_validation.md`
- `reports/decision/latest_runtime_decision_validation.md`
- `reports/controls/latest_negative_control_validation.md`
- `reports/memory/latest_memory_promotion_validation.md`
- `outputs/memory/latest_memory_promotion_report.json`
- recent commits and tags

## Core Rule

```text
theory tells why
architecture tells how
runtime tells now
```

## Version-Readiness Lock

```text
No fresh-thread versioning without Origin Scan, Architecture Scan, and Runtime State Scan.
```

## Non-Claim Lock

Thread rehydration improves agent orientation and repository continuity. It does not prove code correctness, truth, AGI, consciousness, production readiness, security, external validation, or real-world correctness.
