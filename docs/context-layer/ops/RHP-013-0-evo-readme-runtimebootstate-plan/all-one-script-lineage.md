# RHP-013.0 All-One Script Lineage

This file records the governed All-One script style used by James Paul Jackson's Hermes-Agent-Evo work.

## Canonical All-One Shape

```text
Anchor root
-> verify expected root
-> find Python
-> git reset
-> git pull --rebase origin main
-> require exact human authorization phrase
-> execute embedded Python helper from TEMP
-> helper validates prerequisites
-> helper writes bounded candidate files
-> helper writes evidence JSON / closure docs / hashes
-> boundary-check dirty paths
-> git reset
-> git add only allowed paths
-> staged added-line secret trigger scan
-> git commit
-> git pull --rebase origin main
-> git push origin main
-> final status
-> return to root
```

## Required Guards

- Exact human authorization phrase.
- Expected root check.
- Allowed-path boundary.
- Evidence JSON.
- Secret trigger scan over staged added lines.
- No commit/push after failed validation.
- Pull/rebase before mutation and after commit.
- Runtime authority flags remain false unless a future operation explicitly grants a bounded authority class.
